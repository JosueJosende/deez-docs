# Altura con transicion en css

lograr transiciones suaves y atractivas es una de las claves para mejorar la experiencia de usuario. Sin embargo, animar la altura de un elemento en CSS ha sido tradicionalmente un reto, especialmente cuando se desea pasar de un valor fijo a uno `auto`.

Puedes animar la propiedad `height` de manera sencilla y efectiva, utilizando las últimas capacidades de CSS.
Abajo, un ejemplo práctico y cómo implementarlo en tus propios proyectos.

### Código
```css
  article {
    height: 150px;
    overflow: hidden;
    transition: height .5s;
    interpolate-size: allow-keywords;

    &:hover {
      height: auto;
    }
  }
```
